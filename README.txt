# Avant Guardian Boss System
# Author: Shawn Pryde
# Date: 04/31/2021

The scripts included here are the basic components for the
boss system of the game Avant Guardian. The most pertinent 
scripts to observe are "Boss", "AbstractAttack", "Action",
"Factor", and "VariableFactor", But other scripts have been
included should you desire more context. A select few related 
scripts have been omitted as they were neither core components
nor were they authored by myself.
